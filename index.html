Robot Arm

A robot arm system controlled using a joystick and button interface. The VRX axis controls the base rotation (60°–120°), while the VRY axis controls the shoulder movement (60°–150°). The claw is operated by a button, toggling between open and closed states. The system uses Arduino with analogRead() to map joystick inputs to servo angles, allowing real-time manual control for object manipulation.

Embedded System Arduino C Language Joystick Module Servo Motor Button Control

Traffic Light

A traffic light system using LEDs and seven-segment displays for both automatic and manual control. In automatic mode, four-way intersection lights operate on a timed cycle, alternating between green and red like a real traffic light. In manual mode, using a push-button: the first press turns on one LED, the second press turns on two LEDs, until all LEDs light up. Pressing again makes all LEDs blink, and the next press turns them all off.

Embedded System C Language LED Seven Segment Push Button Traffic Light Simulation

Joystick & Web-Controlled LED

A dual-function LED control system using ESP8266. The first mode uses a joystick module connected to ESP8266 to control two LEDs: moving the joystick right lights up the red LED, moving it left lights up the green LED, and pressing the button lights up both LEDs simultaneously. The second mode connects ESP8266 to WiFi, allowing LED control via a web page. The ESP8266 hosts a web server with real-time LED status updates and controls using WebSockets, enabling users to turn LEDs on and off directly from a browser interface.

Embedded System ESP8266 C Language Joystick Module LED Control WebSocket Web Server WiFi

IoT Box by Mattel

IoT Box for Machine Monitoring as part of the Mattel progam, designed for real-time machine data collection in dynamic and mobile manufacturing environments. The IoT Box integrates multiple sensors—fire sensor, DHT22 (temperature & humidity), motion sensor, and vibration sensor (SW-420)—to detect critical machine conditions such as overheating, fire, excessive vibration, and operational status. Sensor data is transmitted via the MQTT protocol to a Firebase Realtime Database, visualized through a web interface, and backed up locally. This system enables predictive maintenance and flexible deployment for smart factory applications, even in areas where wired IoT infrastructure is impractical.

IoT MQTT Firebase Realtime DB DHT22 Fire Sensor SW-420 Motion Sensor Web Dashboard Edge Computing
